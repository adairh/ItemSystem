# ItemSystem
A spigot plugin for minecraft allow to modify what items have been added in to system although they are in players' inventory!
# Features
Add some other Stats like: 
   <li>
    DAMAGE: How much damage item can deal
    MAGICAL_DAMAGE: How much magical damage item can deal
    ARMOR: How much armor value or how much damage can be reduced
    MAGICAL_ARMOR: How much magical armor value or how much magical damage can be reduced
    CRITICAL: How many percentage of critical hit will be multiplied
    CRITICAL_CHANCE: How many percentage that critical hit will happen
    PIERCE: How much damage can deal through armor
    MAGIC_PIERCE: How much magical damage can deal though magical armor
    VAMPIRE: How much health can be absorb by attacker for a single hit
    VAMPIRE_CHANCE: How many percentage that lossing health will be absorb by attacker
  </li>
